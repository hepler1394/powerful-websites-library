# Powerful Free Websites Library

A searchable, installable library of **hundreds of powerful free websites & tools** — AI apps, streaming, gaming, dev tools, VPNs, and more — curated, categorized, and synced with favorites.

![Powerful Free Websites Library](assets/hero.png)

🔗 **Live:** [powerful-websites-library.vercel.app](https://powerful-websites-library.vercel.app)

## Features

- 🔎 Fast search across the full catalog with category filters
- 🗂️ Organized by category with per-category accent theming
- ❤️ Save favorites, synced via **Firebase**
- 🌌 Dark glassmorphism UI with an animated aurora background
- 📱 Installable **PWA** with offline support
- 📄 Companion pages: Best Practices, FAQ, Tools, Onboarding

## Tech

JavaScript · Firebase · Service Workers (PWA) · HTML5 / CSS3
